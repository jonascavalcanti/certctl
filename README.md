# cert-openssl-generator
Generate your Openssl Certificate 

## Command for generate a new Openssl certificate

```
certOpensslGenerator.sh ${CERTS_DIR} \
                            ${COUNTRY} \
                            ${STATE}   \
                            ${LOCALITY} \
                            ${ORGANIZATION} \
                            ${ORGANIZATIONALUNIT} \
                            ${COMMON_NAME} \
                            ${EMAIL}

```
